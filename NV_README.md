# Libcoap for Northvolt Power Electronics

Libcoap version v4.3.4 is used

The following files are excluded from build:

* `coap_asn1.c`
* `coap_event.c`
* `coap_io_contiki.c`
* `coap_io_riot.c`
* `coap_mbedtls.c`
* `coap_mem.c`
* `coap_openssl.c`
* `coap_oscore.c`
* `coap_time.c`
* `coap_tinydtls.c`
